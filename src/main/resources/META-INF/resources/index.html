Oups !

You need to generate the frontend part of the project

How-to :

  • Make sure that npm v6.9.0+ is installed
  • go to the hotel-quarkus folder
  • type-in 
    npm run build
    this will compile and minify the frontend part for production. It should generate the `hotel-quarkus/dist` folder
  • copy the dist folder to the src/main/resources/META-INF/resources folder
    cp -R dist/* ../src/main/resources/META-INF/resources/
  • reload this page, and since the index.html is overwritten, you should now see the Quarkus hotel demo
